พบปัญหา Failed to fetch (CORS Error)

ปัญหานี้เกิดขึ้นเนื่องจาก Google บล็อกการเชื่อมต่อเพื่อความปลอดภัย สาเหตุที่เป็นไปได้มีดังนี้:

1คุณเปิดไฟล์ HTML โดยตรง (file:///)

เบราว์เซอร์ไม่อนุญาตให้ไฟล์ที่เปิดจากเครื่องตรงๆ เรียกใช้งาน API

วิธีแก้: ต้องรันผ่าน Local Server เช่น ใช้ Extension Live Server ใน VS Code หรือใช้คำสั่ง python -m http.server

2ยังไม่ได้ตั้งค่า Authorized Origins ใน Google Cloud

ถ้าคุณรัน Local Server แล้วได้ URL เช่น http://localhost:5500 หรือถ้าคุณรันบนเว็บอื่นๆ

วิธีแก้: คุณต้องเข้าไปที่ Google Cloud Console ไปที่ Client ID ของคุณ และเพิ่ม URL ปัจจุบันนี้ ลงในช่อง Authorized JavaScript origins จากนั้นรอประมาณ 5 นาที

3Access Token หมดอายุ

Token ของ Google มีอายุเพียง 1 ชั่วโมง เมื่อหมดอายุการใช้งาน Google จะไม่ส่งค่าอนุญาตกลับมา ทำให้เกิด Error นี้เช่นกัน

วิธีแก้: กลับไปหน้าตั้งค่า และขอ Access Token ชุดใหม่มาใส่
